IoT-Based Crack Detection and Monitoring System

Project Description

This project develops an Internet of Things (IoT)-based crack detection and monitoring system for buildings. Wireless sensors are installed at critical points to detect structural cracks and send data to a cloud platform. The system provides real-time monitoring and alert notifications, improving safety and reducing maintenance costs.

Aims and Objectives

  • Design a wireless crack detection system that monitors structural faults in real time.
  • Achieve high accuracy in identifying structural cracks in buildings.
  • Implement affordable IoT sensors for continuous monitoring and alerts.

Methodology

The project uses Agile methodology, which includes six iterative phases: Planning, Design, Development, Testing, Deployment, and Review. This ensures flexibility and continuous improvement of the IoT monitoring system.

Significance

By enabling continuous, real-time monitoring, this system reduces dependency on manual inspections and human error. It enhances safety for occupants, minimizes maintenance costs, and allows data-driven decision-making for infrastructure management.
